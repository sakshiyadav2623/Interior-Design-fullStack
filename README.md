# 🏡 Sahacarah Interior Website

A modern and responsive website for showcasing interior design and construction services, including modular solutions, residential projects, and commercial spaces.

🔗 Live Website: https://sahacarah.in/

---

## 📌 Features

* 🏠 Interior design services (home & commercial)
* 🍽️ Modular kitchen and furniture solutions
* 🏢 Office and workspace design showcase
* 🖼️ Project gallery for design inspiration
* 💰 Pricing and consultation section
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Design:** Responsive UI/UX
* **Hosting:** Web hosting platform Hostinger.

---

## ⚙️ Overview

This website represents a complete digital presence for an interior design business. It allows users to explore services, view design portfolios, and understand pricing and consultation processes.

The platform is designed to provide a smooth user experience while highlighting creativity, professionalism, and modern design trends.
